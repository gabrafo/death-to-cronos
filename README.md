# Death to Cronos ⏱️

## Português

### Sobre o Projeto

Um timer Pomodoro com temática inspirada no vilão Chronos do jogo Hades 2. Este é um projeto pessoal desenvolvido para fins de aprendizagem e prática com React e TypeScript.

### Tecnologias Utilizadas

- React
- TypeScript
- Vite
- CSS Modules

### Status do Projeto

⚠️ **Projeto em desenvolvimento ativo**

Este é um projeto de aprendizagem e ainda há muito a avançar tanto no desenvolvimento quanto no design. Novas funcionalidades e melhorias estão sendo implementadas continuamente.

### Como Executar

```bash
# Instalar dependências
npm install

# Executar em modo de desenvolvimento
npm run dev
```

### Objetivo

Desenvolver um aplicativo Pomodoro funcional enquanto aprimoro minhas habilidades em:
- Gerenciamento de estado com React
- TypeScript
- Componentização
- Estilização com CSS Modules
